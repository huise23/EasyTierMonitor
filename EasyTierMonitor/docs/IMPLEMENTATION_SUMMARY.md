## 🔧 技术细节

### 配置对话框实现要点

1. **数据绑定**
   - 构造函数接收当前配置和 Peer 列表
   - OnInitDialog 加载配置到控件
   - OnOK 验证并保存配置

2. **输入验证**
   - 更新间隔: 1-60 秒
   - Ping 超时: 500-5000 毫秒
   - 失败阈值: 1-10 次
   - 通知冷却: 1-60 分钟
   - 历史记录: 10-1000 条

3. **配置应用**
   - 配置更改立即保存到文件
   - 工作线程自动重启以应用新配置
   - DataManager 线程安全更新

### 通知管理器实现要点

1. **冷却机制**
   ```cpp
   bool CanSendNotification() {
       double diff = difftime(current_time, last_notification_time_);
       return diff >= cooldown_minutes_ * 60;
   }
   ```

2. **平台适配**
   - Windows 10+: 使用 Toast 通知
   - Windows 8-: 使用消息框回退

3. **快捷方式创建**
   - 创建开始菜单快捷方式（预留）
   - AUMID 注册（预留）

### 历史记录对话框实现要点

1. **数据展示**
   - 列表框显示状态变化时间线
   - 倒序显示（最新的在上面）
   - 图标 + 时间 + 状态 + 延迟

2. **统计计算**
   ```cpp
   - 总记录数
   - 各状态计数
   - 平均延迟 (仅成功的 Ping)
   ```

3. **导出功能**
   - CSV 格式，UTF-8 编码
   - 包含 BOM 标记
   - 时间、状态、延迟、消息

## 💡 使用说明

### 配置插件
1. 打开 TrafficMonitor
2. 右键 → 显示设置 → 插件设置
3. 选择 "EasyTier Peer Monitor"
4. 点击 "选项" 按钮打开配置对话框

### 查看历史记录
1. 在 TrafficMonitor 中右键插件显示项
2. 选择 "查看历史记录" (功能待添加)

### 导出历史记录
1. 在历史记录对话框中
2. 点击 "导出..." 按钮
3. 选择保存位置和格式
