# EasyTier Monitor - 更新日志

## [0.1.1] - 2026-03-15

### 新增功能
1. **配置对话框**
   - 添加了简化的配置界面
   - 支持选择固定 peer 或自动轮换模式
   - 可以刷新 peer 列表

2. **名称简化**
   - 自动去除 `PublicServer_` 前缀
   - 显示更简洁的 peer 名称

3. **Ping 功能增强**
   - 添加详细日志记录
   - 优化 ping 执行时机
   - 确保 ping 结果正确显示

### 修复问题
- 修复了 ping 延迟显示为 `-` 的问题
- 优化了配置对话框的用户体验

### 技术改进
- 添加了 `auto_rotate_` 和 `selected_peer_index_` 配置选项
- 简化了资源文件，移除了不必要的控件
- 改进了日志输出，便于调试

## [0.1.0] - 2026-03-13

### 初始版本
- CLI 调用和解析
- 表格数据解析
- 滚动显示所有 peer
- ICMP Ping 延迟测量
- 四项信息显示：hostname | cost | tunnel_lat | ping_lat
- 日志系统
